CSS Units and Measurments

CSS provides various units to define dimensions, spacing and typography. These units are categorised into absolute and relative types

1. Absolute units

Absolute units define fixed sizes that do not change relative to the viewport or parent elements

  • px - Pixels
  • cm - Centimetes
  • mm - Millimetres
  • in - Inches

2. Relative units --- Responsive design

Relative Units depend on the parent elements, viewport size or root elements, making them ideal for responsive design

em and rem are font related units. em unit is relative to the parent element font size and rem is relative to the root element (HTML) font size

Percentage(%) units are relative to the parent element (width or height)

vw (viewport width) and vh (viewport height) units are relative to the viewport/available screensize
