CSS- Cascading Styles Sheets

CSS is a programming language used to improve the presentation of a html page

CSS will help modify aspects like texts sizes, colour , backrounds, image sizes and shapes, borders positions, responsiveness (different looks on different screen sizes) among other aspects of presentation

CSS is used to design user interfases for web pages and it helps o enhance user experience

CSS syntax (how do we write CSS)

A selector is used to target an element or a group of elements for styling

{} -A declaration block - groups properties and values to be applied on the selector

Property - a presentation aspect you'd want to modify eg color, width , size, position

Value- the final look you are interested in for the page (red, fullpage, 400px ...) 

         
            selector{
                property:value;
                property:value;
            }
         
         
Example

         
            h1{
                color:green;
            }

Where do we put our CSS

  • Inline CSS

    We use the style element on individual elements.(Code is less reusable ) and makes maintaining the application difficult

  • Internal CSS

    Using the style element .(Doing styles within same files as structure is not recommended.Styles won't be shared to other HTML pages.)

  • External CSS (Recommended)

    Using a different file ( .css) linked to the html page using the link elementusually placed in the head of a html page.
